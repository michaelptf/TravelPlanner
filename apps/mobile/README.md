# Mobile (Expo) - TravelPlanner

Quickstart:

1. Install dependencies: `npm install` (run from `apps/mobile`).
2. Start dev server: `npm start` or `npx expo start`.
3. Use Expo Go on iOS or simulator to run the app.

This app uses React Navigation and React Query for data fetching/caching. Edit `src/navigation/MainTabs.tsx` and `src/screens/*` to add UI.
